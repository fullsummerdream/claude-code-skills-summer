# 公文文风包

> 涵盖：通知、报告、请示、函、纪要。
> 公文文风的本质不是"格式正确"，而是**以组织名义说话，让任何经手人都能无歧义地执行**。
> 写公文时，个人彻底退场，机构出场；文字不追求好读，追求不出错、不歧义、可追溯。

## 角色与读者

- 我是谁：发文机关/单位的执笔人。落款是"XX办公室""XX公司行政部"，不是张三李四。全文代表机构意志，不代表个人观点。
- 读者是谁：上级机关（请示、报告）、平级或不相隶属单位（函）、下级单位与全体员工（通知、纪要）。读者读公文是为了**明确"要我做什么、何时完成、依据是什么"**，不是为了获得阅读体验。
- 与读者的关系：组织层级内的公务关系。上行文（请示/报告）有明确的服从与请示姿态；平行文（函）对等客气但不熟络；下行文（通知/纪要）明确指令、不留商量余地。

## 深层文风特征（比格式更重要的东西）

1. **作者的隐身**。公文的最高准则是"看不出是谁写的"。同一份通知由甲写还是乙写，成品应当无差别。因此：不追求个人表达、不展示文采、不留下风格指纹。判断一段文字像不像公文，第一眼看的就是"这段话里有没有一个具体的人在说话"。
2. **可执行性优先于可读性**。公文写完要过一道自检：一个完全不了解背景的经手人，只凭这段文字，能否明确知道"做什么、谁做、何时完成、做完交给谁"？答不上来的部分就是文风事故，不是格式问题。
3. **零歧义是硬指标**。每一个代词、每一个时限、每一个范围表述（"各部门""相关单位""原则上"）都须经得起追问。"原则上同意"在公文里是特定术语（=同意但细节待核），日常语境里却是敷衍——公文用词有自己的语义系统，不能按口语理解。
4. **语气的层级感**。公文的三套语气（对上的恳而不卑、对平的客气对等、对下的明确指令）是文风的有机部分，不是客套。对上级说"责令"、对下级说"盼复"，都是方向性错误。
5. **程式不是僵化，是契约**。"特此通知""妥否，请批示"这类固定结尾，作用是让读者一眼就定位文种与效力。自创结尾（"让我们携手共进"）等于撕毁契约。

## 写作规则

### 基础规则

1. **一文一事**。一份文件只说一件事。请示不得夹带报告事项，通知不得混入两项不相干的安排。
2. **标题三要素齐全**：发文机关 + 事由 + 文种。例：《XX公司关于开展年度安全检查的通知》。"关于"引出事由，文种（通知/请示/报告/函/纪要）收尾。
3. **开头程式化**，直接交代依据和目的，第一段说完"为什么发这份文"：
   - 通知："为……，现将有关事项通知如下。"
   - 请示："因……，现请示如下。"
   - 报告："根据……要求，现将有关情况报告如下。"
   - 函："贵单位来函收悉。经研究，现函复如下。"
   - 纪要："X年X月X日，XX在XX会议室主持召开……会议。现纪要如下。"
4. **主体分条列项**：两项以上内容一律用"一、""二、""三、"或"（一）（二）（三）"分层，一级标题用汉字数字加顿号，二级用括号汉字数字，不用"1.2.3"式科技编号（党委政府公文惯例）。
5. **结尾程式化**，按文种固定：
   - 通知："特此通知。"（下行可加执行要求："请遵照执行。"）
   - 请示："妥否，请批示。"或"以上请示，请予审批。"
   - 报告："特此报告。"
   - 函："特此函复。"/"盼复。"
   - 纪要：无固定结语，以最后一条议定事项自然结束。
6. **时间与数字精确到点**："7月21日14:00前"，不写"下周左右""尽快""近期"。模糊时限是公文最常见的执行事故来源。
7. **责任主体明确到单位/岗位**：每条事项写清"谁做"。"由行政部牵头，各部门配合"，不写"有关方面要抓好落实"。
8. **依据可溯**：引用上级文件须写全文件名与文号，如"根据《XX管理办法》（X发〔2026〕12号）"。
9. **语气按行文方向调节**：上行文（请示/报告）陈述而不恳求、不施压；下行文（通知）明确指令，用"要""须""不得"；平行文（函）对等，用"请贵单位""盼复"，不用"责令""务必"。
10. **请示必须事前行文、只写一个主送机关**；报告可在事中事后，且报告中不得含请示事项。

### 禁止词/禁止句式

**禁止的词与短语：**

- 一切口语词：咱们、搞定、差不多、回头、到时候、弄、整、呗、哈、呢（句末语气词）。
- 一切网络与营销用语：赋能、抓手、闭环、颗粒度、组合拳、打 call、刷屏、爆款、重磅、燃。
- 一切情绪与评价副词：非常、十分（可用"较""进一步"替代）、极其、令人振奋、值得骄傲、可喜可贺（贺信类专用除外）。
- 模糊时限词：尽快、近期、适时、届时（对外行文须给具体日期；内部确有未定事项可用"另行发文明确"）。
- 第一人称个人抒情：我认为、我觉得、我感到十分荣幸、本人谨代表……（落款个人签名场景除外）。
- 比喻与修辞性意象：像……一样、犹如、翻开了新的一页、吹响了号角、扬帆起航、迈上新台阶（后三者为新闻通讯用语，公文不用）。

**禁止的句式：**

- 感叹句、反问句、设问句：一个都不许出现。
- 排比抒情句："这是……的时刻，这是……的时刻"。
- 对话式、呼告式："各部门，你们要……"
- 悬念式开头："为什么会这样？请看下文。"
- 主观推测句：可能、大概、也许会——确需表达不确定性时用"初步判断""尚待核实"并注明信息来源。

### 句式偏好

- **陈述句为绝对主体**，判断句次之。基本骨架：主语（单位/事项）+ 谓语（开展/落实/报送/予以）+ 宾语。
- **"为……，现将……如下"**：目的前置 + 引出下文的万能开头骨架。
- **"经……（研究/审议/讨论），决定……"**：决议类表述骨架，决策过程一笔带过。
- **"请……于X月X日前……，并将……报送至……"**：通知任务条目的标准骨架——指令 + 时限 + 反馈方式，三要素缺一不可。
- **"会议听取/审议/研究了……，议定/认为/要求……"**：纪要条目骨架，每一条以动词领起。
- **无主句可用但主语须可从上下文唯一确定**；不得出现指代不明的"该""此""其"（每处代词回读时须能立刻找到唯一先行词）。
- 长句拆分：单句超过 40 字或含三层以上修饰时，拆为两句或改为分条。公文允许句子"笨"，不允许句子"绕"。
- 并列成分用顿号或分号分层，不堆"和"："负责场地、设备和后勤保障"，不写"负责场地和设备还有后勤保障"。

### 常用公文用语库（可直接取用）

| 场合 | 用语 |
|---|---|
| 开头表依据 | 根据、按照、遵照、为落实……要求 |
| 开头表目的 | 为……，进一步加强……，切实…… |
| 承接引出 | 现将有关事项通知如下／现纪要如下／经研究 |
| 指令性动词 | 开展、组织、落实、报送、排查、整改、予以、严格执行 |
| 程度与限定 | 进一步、切实、从严、原则上（=同意但细节待核）、酌情 |
| 上行文结尾 | 妥否，请批示／特此报告／以上意见如无不妥，请批转 |
| 平行文结尾 | 特此函达／盼复／特此函复／请予支持为盼 |
| 下行文结尾 | 特此通知／请遵照执行／请抓好贯彻落实 |

> 用词库的边界：这些用语保证稳妥，但不能替代内容。骨架正确、事项空泛（通篇"切实提高认识、加强组织领导"）仍是坏公文——**程式管形式，基础规则 6/7/8 管实质，两头都要有**。

## 格式规范

- **段落长度**：每段一个层次。开头段 2-4 句（依据+目的+过渡句）；主体以条列项，每条 1-3 句，超过 5 句的条应再分（一）（二）。不写大段密集文字。
- **标题层级**：一级"一、"，二级"（一）"，三级"1."，四级"（1）"，层级不得跳用，最多四级。一级小标题末尾不加标点或加句号（惯例从严：不加）。
- **文号与落款**：正式行文有发文字号（X办发〔2026〕X号，年份用六角括号）；落款为发文单位全称 + 成文日期（阿拉伯数字：2026年7月21日）。
- **主送与抄送**：主送机关在标题下顶格，后加冒号；请示只主送一个机关。抄送机关列于文末。
- **数字用法**：成文日期、统计数字、文号中的序号用阿拉伯数字；结构层次序数（一、（一））用汉字；概数用汉字（"三四个"）。年份不缩写（"2026年"不写"26年"）。
- **标点**：标题一般不用标点（法规性文件名可用书名号）；条列项内句末用句号；不用感叹号、问号、省略号。
- **附件**：有附件时在正文末段前注明"附件：1.XXXX表"，附件本身另页编排。
- 不使用 Markdown 语法符号（#、**、- 列表）冒充公文格式；以文本交付时用汉字序号体系替代。

>
> 一、放假时间。10月1日（星期四）至10月7日（星期三）放假调休，共7天。9月27日（星期日）、10月10日（星期六）正常上班。
>
> 二、值班安排。各部门须于9月25日前将值班表报送行政部备案；值班人员须保持通信畅通，遇突发事件按《突发事件应急处置预案》（X司发〔2025〕8号）处置并即时上报。
>
> 三、安全要求。各部门于放假前完成办公区域安全检查，关闭非必要电源设备。

## 自检清单（写完公文过一遍）

1. 全文能否找到一处口语词、一个感叹号、一个比喻？——能就删。
2. 每条任务能否回答"谁做、何时完成、交给谁"？
3. 每一处"该/此/其"回读，先行词是否唯一？
4. 引用的文件是否都有全称和文号？
5. 开头第一段是否已交代依据与目的？结尾是否用了所属文种的固定结语？
6. 行文方向是否对：请示有没有夹带报告事项？函里有没有出现"责令/务必"？通知里有没有出现"盼复"？
7. 把落款单位遮掉，这段文字像不像"任何一家正规单位都可能发出的文"？——像，就对了。

## 各文种一句话区分

- **通知**：让下级或全员知道并执行某安排——指令 + 时限 + 反馈方式。
- **报告**：向上级汇报情况，只陈述，不求批复，不夹带请示事项。
- **请示**：事前请求上级批准，一文一事一个主送，结尾"妥否，请批示"。
- **函**：与不相隶属单位商洽、询问、答复——对等客气，"盼复""特此函复"。
- **纪要**：记录会议议定事项，以"会议认为/议定/要求"领起，不记录个人发言原话，不评论。
